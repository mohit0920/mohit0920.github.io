CSS

cascading style sheets. CSS works by defining rules how a selector(html tag) can be displayed. A CSS rule set = selector+declaration block

​x
Selector{property:value;property:value;}

Grouping Selectors

Various selectors can be grouped with comma. 

xxxxxxxxxx
h2, h3, p
{
  color:blue;
  font-weight:italics;
}

Selectors

id selector

Id Should not start with a number.

xxxxxxxxxx
#p1, h1 
{
  text-align:left;
  color:red;
}

Class Selector

name of class can be char, string but can not start with Number.

xxxxxxxxxx
p.center, tr.center
{
  text-align:center;
  color:pink;
}

above ruleset is just applied to para and table row elements with class center.

Class can be styled without any element or html tag. where all elements of same class are style together.

xxxxxxxxxx
.right
{
  text-align:right;
  color:blue;
}

ELement Selector

xxxxxxxxxx
p{
  text-align:left;
  color:grey;
}

Inserting CSS into Html

External Style Sheet

ideal when applying style to many pages. Can link to html using link tag under the head tag.

xxxxxxxxxx
<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

The import can also be used to import an external css file.

xxxxxxxxxx
<head>
<style type="text/css">
  @import url(mystyle.css);
</style>
</head>

Internal style sheet

Are used when aplying unique styles to single documents. Written inside head tag using style tag.

xxxxxxxxxx
<head>
<style>
.left {
  color:blue;
}
</style>
</head>

Inline Styles

written as a part of html tag itself with style attribute. Its least flexible. 

xxxxxxxxxx
<p style="color:blue;text-align:left">
  this is a paragraph
</p>

Default preference

Inline>Internal>External

Overriding preference

for example if you want to push external styling for an element.

x
h3 
{
  color: blue !important;
}
